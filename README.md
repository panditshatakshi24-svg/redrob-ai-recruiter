# 🤖 Hybrid AI Candidate Ranking System

An AI-powered candidate ranking system built for the Redrob India Data & AI Challenge.

## Features

- Hybrid AI Ranking
- Feature Engineering
- Semantic Search
- Explainable AI
- Streamlit Dashboard
- CSV Export

## Tech Stack

- Python
- Sentence Transformers (BAAI/bge-small-en-v1.5)
- Scikit-learn
- Pandas
- Streamlit

## Live Demo

https://your-streamlit-app.streamlit.app

## Architecture

Job Description
↓
Keyword Extraction
↓
Feature Engineering
↓
Top 300 Retrieval
↓
Semantic Re-ranking
↓
Top 100 Candidates
